# 🎓 دليل رفع حقيبة الترقية على GitHub (عرض فقط بلا تحميل)

الهدف: استضافة حقيبة الترقية (187 ملف) على GitHub، وعرضها في المكتبة
بعارض خاص **بلا زر Google وبلا تحميل**.

---

## الخطوة 1 — نزّل حقيبة الترقية من Google Drive
1. افتح مجلد **حقيبة الترقية** في Drive
2. فوق يمين، اضغط على النقاط الثلاث ⋮ → **تنزيل / Download**
3. Drive راح يضغطها في ملف ZIP — نزّله
4. فك الضغط (Extract) — راح يظهر مجلد **حقيبة الترقية** بداخله المجلدات الفرعية

---

## الخطوة 2 — أنشئ حساب ومستودع GitHub (بلا بطاقة)
1. روح إلى **github.com** → **Sign up** (إن لم يكن عندك حساب)
2. اضغط **+** فوق يمين → **New repository**
3. الاسم: `tarqia` (أو أي اسم بحروف صغيرة إنجليزية)
4. اختر **Public** ✅ (مهم)
5. فعّل **Add a README file**
6. اضغط **Create repository**

---

## الخطوة 3 — ثبّت GitHub Desktop
1. روح إلى **desktop.github.com** → **Download**
2. ثبّت البرنامج وسجّل الدخول بحسابك (نفس حساب GitHub)

---

## الخطوة 4 — استنسخ المستودع (Clone)
1. في GitHub Desktop: **File → Clone repository**
2. اختر مستودع `tarqia` → **Clone**
3. اختر مكانًا على حاسوبك يُحفظ فيه (احفظ المسار)

---

## الخطوة 5 — انسخ الملفات وارفعها
1. افتح مجلد المستودع المستنسخ (الذي اخترته في الخطوة 4)
2. انسخ إليه مجلد **حقيبة الترقية** كاملاً (بمجلداته الفرعية كما هي)
   - أي المسار يصبح: `tarqia/حقيبة الترقية/التّحضير/...`
3. ارجع إلى **GitHub Desktop** — راح يظهر لك 187 ملف جديد
4. تحت (Summary) اكتب: `add tarqia files`
5. اضغط **Commit to main**
6. فوق اضغط **Push origin** (لرفعها إلى GitHub)
7. انتظر اكتمال الرفع (قد يأخذ دقائق حسب النت — 241 ميغا)

---

## الخطوة 6 — عدّل رابط الملفات في قاعدة البيانات
1. افتح ملف `insert_tarqia_github.sql`
2. استبدل **USER/REPO** باسم حسابك ومستودعك.
   مثال: حسابك `sacsvtprof`، المستودع `tarqia`:
   - من: `cdn.jsdelivr.net/gh/USER/REPO@main/...`
   - إلى: `cdn.jsdelivr.net/gh/sacsvtprof/tarqia@main/...`
   (استعمل Replace All في أي محرّر)
3. **مهم:** إن كنت قد شغّلت `insert_tarqia.sql` (روابط Drive) من قبل،
   أزل التعليق عن سطر الحذف في أول الملف لمسح القديم:
   `delete from public.files where category='الترقية';`
4. انسخ الملف كاملاً → Supabase → **SQL Editor** → **Run**

---

## ✅ النتيجة
افتح المكتبة → قسم «حقيبة الترقية» → اضغط «عرض»:
- الملف يُعرض بعارض خاص **بلا زر Google** ✅
- **بلا زر تحميل** ✅ (عرض فقط)

---

### 💡 ملاحظات
- **ملفات Word (14 ملف):** لا تُعرض في عارض PDF.js. الأفضل تحويلها إلى PDF قبل الرفع
  (افتحها في Word → حفظ باسم → PDF)، أو تركها (ستظهر لكن لن تُعرض).
- **العروض PowerPoint (20 ملف):** كذلك تحتاج تحويل إلى PDF لتُعرض بالعارض الخاص.
- **jsDelivr** قد يأخذ بضع دقائق لتحديث الملفات الجديدة أول مرة.
- للتحديث لاحقًا: أضف ملفات في نفس المجلد، Commit، Push.
