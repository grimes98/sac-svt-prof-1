# ☁️ دليل إعداد Cloudflare R2 (مرة واحدة) — لتخزين 200 مذكرة مجانًا

R2 يعطيك **10 جيغا مجانًا + تحميل بلا رسوم** — مثالي للمكتبة.
اتبعي الخطوات بالترتيب. تحتاجينها **مرة واحدة فقط**.

---

## الخطوة 1 — أنشئي حساب Cloudflare
1. روحي إلى **dash.cloudflare.com/sign-up**
2. أدخلي بريدًا وكلمة مرور → تأكيد البريد

---

## الخطوة 2 — فعّلي R2
1. من القائمة اليسرى اضغطي **R2 Object Storage**
2. راح يطلب تفعيل — اضغطي **Purchase R2 Plan** / **Enable**
   > ⚠️ لا تخافي: الخطة المجانية (10 جيغا) بلا بطاقة في معظم الحالات. إذا طلب بطاقة، لن يُخصم شيء ضمن الحد المجاني.

---

## الخطوة 3 — أنشئي حاوية (Bucket)
1. اضغطي **Create bucket**
2. الاسم: `sac-library` (أو أي اسم بحروف صغيرة)
3. **Location**: اتركيه Automatic
4. اضغطي **Create bucket**

---

## الخطوة 4 — فعّلي الوصول العام (r2.dev)
1. ادخلي للحاوية → تبويب **Settings**
2. ابحثي عن **Public access** / **R2.dev subdomain**
3. اضغطي **Allow Access** → أكّدي (اكتبي "allow")
4. راح يظهر رابط عام مثل:
   `https://pub-xxxxxxxx.r2.dev`
   **انسخيه واحتفظي به** — هذا رابط ملفاتك العام.

---

## الخطوة 5 — أنشئي مفاتيح API (Access Keys)
1. ارجعي لصفحة **R2** الرئيسية
2. على اليمين اضغطي **Manage R2 API Tokens** (أو **API** → **Create API token**)
3. اضغطي **Create API token**
4. الصلاحيات (Permissions): اختاري **Object Read & Write**
5. اضغطي **Create API Token**
6. راح تظهر لك **مرة واحدة فقط** (انسخيها كلها فورًا):
   - **Access Key ID**
   - **Secret Access Key**
   - **Endpoint** (رابط مثل: `https://<account_id>.r2.cloudflarestorage.com`)

> 🔒 احفظي هذه المفاتيح في مكان آمن — الـ Secret لن يظهر مرة أخرى.

---

## ✅ بعد إتمام الخطوات، جهّزي هذه القيم الخمس:

| القيمة | مثال | من أين |
|---|---|---|
| **Account ID** | `a1b2c3...` | من الـ Endpoint أو صفحة R2 |
| **Access Key ID** | `f4e5d6...` | الخطوة 5 |
| **Secret Access Key** | `xY9z...` | الخطوة 5 |
| **Bucket name** | `sac-library` | الخطوة 3 |
| **Public URL** | `https://pub-xxxx.r2.dev` | الخطوة 4 |

➡️ **أعطيني هذه القيم** (ما عدا الـ Secret إن أردتِ — تضعينه أنتِ في السكريبت)، وأنا أجهّز لك سكريبت Colab يرفع الـ200 مذكرة تلقائيًا.

---

### 💡 ملاحظة أمان
- **Access Key + Secret** = مفاتيح حسّاسة (مثل كلمة السر). لا تنشريها علنًا.
- في السكريبت نضعها كمتغيّرات، وتبقى على Colab فقط أثناء التشغيل.
- **Public URL (r2.dev)** = عادي أن يكون ظاهرًا (هو رابط التحميل للأساتذة).
